# 🔧 函數與 Agent 文檔
生成時間：2026-04-05 01:26:50

## 系統 Agent 架構

### Builder
**職責**：準備環境、克隆 repo、下載依賴

### Trainer
**職責**：執行 3DGS 訓練（可選，需要數據）

### Tester
**職責**：執行測試、驗證功能

### Analyzer
**職責**：分析錯誤、分類問題類型

### Fixer
**職責**：應用修復、調整參數

### Critic
**職責**：質疑決策、防止無限 loop

### DocAgent
**職責**：自動生成文檔、記錄進度

## 核心功能

- **事件驅動**（EventBus）：Agent 間通信
- **狀態管理**（State）：全局狀態追蹤
- **自動修復**：檢測→分析→修復→重試
- **自我反思**（Critic）：質疑決策、改進策略
